## 🌟 Featured: the StreamSnip ecosystem

**[StreamSnip](https://github.com/SurajBhari/streamsnip)** is my flagship — it lets a streamer's viewers clip a live moment with a single Nightbot chat command, no clipping software needed. It grew into a small ecosystem:

| Project | What it does |
|---------|--------------|
| 🎬 [streamsnip](https://github.com/SurajBhari/streamsnip) | The product — `!clip` in chat → timestamped clips on streamsnip.com. |
| 🧩 [streamsnip_extension](https://github.com/SurajBhari/streamsnip_extension) | Chrome extension that draws clip markers on the YouTube scrubber. |
| ⬇️ [streamsnip_downloader](https://github.com/SurajBhari/streamsnip_downloader) | CLI that pulls clip segments as real video via yt-dlp. |
| 📊 StreamStats | Real-time YouTube live-chat analytics (Flask + PostgreSQL + Three.js). |
